install with nodejs

```shell
npm install -g gitbook-cli
```

start server 

```shell
gitbook serve --port
```

run in process

```shell
setsid gitbook serve
```

create readme.md and summary.md

```shell
gitbook init
```

build

```shell
gitbook build
```


install mermaid
```shell
npm install gitbook-plugin-mermaid
```

book.json

```json
{
    "plugins":["mermaid-gb3"]
}
```

install plugin

```shell
gitbook install
```
